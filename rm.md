rm file.txt - will remove that file
rm ~/directory/file- will remove that file from the mentioned directory in the home directory.
rm -r directory - will remove that directory recursively.
rm -f directory -will remove that directory forcibly without conforming with the owner.
rm -rf directory - will recursively and forcely remove that directory.
rmdir directory - will remove empty directory.
rm -i file.txt - will remove the file but before removing will confirm with the owner as "y" for yes and "n" for no.
