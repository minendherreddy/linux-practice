ls- list all the files and subdirectories in the current directory
cd- change the directory
whoami- gives the user information
echo "hai" - will print what ever there in the inverted commas
id- will give information like userid,groupid etc
touch filename - will create a file
cat filename- will read that file
echo "hello what's your name?" > new.txt - will write that content into that file and if the file is not created it will create that file and write that content
cd ~ - will go to home directory
cd ~/Desktop- will go to Desktop in home directory
