<img width="1919" height="1008" alt="image" src="https://github.com/user-attachments/assets/4e76cc87-8dc4-4e88-bd95-7bebca09320b" />ls- list all the files and subdirectories in the current directory
cd- change the directory
whoami- gives the user information
echo "hai" - will print what ever there in the inverted commas
id- will give information like userid,groupid etc
touch filename - will create a file
cat filename- will read that file
echo "hello what's your name?" > new.txt - will write that content into that file and if the file is not created it will create that file and write that content
cd ~ - will go to home directory
cd ~/Desktop- will go to Desktop in home directory
echo "this is hidden file" > .hidden.txt - will write that content into  the .hidden.txt file and if the file is not created before it will create that file and write that content and file is hidden
ls -a - list all the files including hidden files
ls -l ~ - will print all the files in the home directory will additional information like file permission and owner,group last updated time etc.
ex: drwx-rw-r kali kali 12 may 23 12:30 Desktop
mkdir MINNU - will create that directory "MINNU"
